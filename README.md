# web-ttt

A game of tic tac toe playable in a browser (in progress).

## Installation

Clone this repo to your machine.

## Usage

Require leiningen and JDK 

```
lein compile
lein uberjar
java -jar target/uberjar/web-ttt-0.1.0-SNAPSHOT-standalone.jar
```
